# Android_mytetris
俄罗斯拼图游戏：
俄罗斯方块的游戏形式，拼图游戏的游戏玩法。

1.0版本运行展示：
1.1版本，新增上传图片功能，玩家能上传本地图片作为拼图背景
2.0.0版本：生成apk文件，喜欢的朋友可以下载游玩，但需要在打开前手动给权限，因为没有写动态权限申请（等有空了完善一下）
